SAE 2.04
Exploitation d'une base de données créée à base de données libres.

Le but de cette SAE est de créer une base de données en exportant des données libres que l'on trouve sur internet.

Il fallait donc choisir un ou plusieurs jeux de données sur le sujet de notre choix, les trier, et les traiter pour ensuite les importer dans la base de données afin de les analyser.

Pour cela j'ai cherché une vingtaine de jeux de données sur le thème du cinéma.

J'ai ensuite participé au tri et au traitement des données en Python pour garder seulement celles qui nous intéressaient.

Et enfin, j'ai contribué à l'analyse de ces données directement dans notre base de données.

SAE 2.01
Création d'un jeu fonctionnel

Le but de cette SAE est de créer un jeu fonctionnel en créant tout d'abord une partie visuelle, puis une partie fonctionelle et enfin lier ces deux parties ensemble.

Pour la partie visuelle j'ai réalisé le plateau de jeu en XAML.

J'ai ensuite participé à la partie fonctionnelle en créant la boucle de jeu et l'affichage du plateau sur un terminal en C#.

Et enfin, j'ai contribué à la liaison de ces deux parties grâce au binding.
